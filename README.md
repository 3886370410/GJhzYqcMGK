# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
- 协同过滤算法
- ECharts可视化

## 适用场景
- 社区便民服务
- 数据分析
- 用户行为研究

## 技术特点
- 采用B/S架构，支持MVC模式
- 使用SpringBoot和Vue构建
- 开发环境包括IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈包括Java、SpringBoot、Vue
